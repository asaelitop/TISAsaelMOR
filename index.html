Proyecto Integrador

Módulos

Este proyecto consiste en el desarrollo de un sistema para la gestión de una cafetería, integrando módulos de Análisis de ventas, Marketing, Personal, Ventas, Ingredientes, Proveedores, Clientes, Inventario, Facturación, Carta, Seguridad, Membresía.

Problemática

Actualmente, la cafetería no cuenta con un sistema que administre sus procesos de manera integral, lo que genera problemas de control en inventarios, ventas y reportes.

Objetivo

Desarrollar un sistema integral que optimice la gestión de productos, ingredientes, proveedores y ventas, mejorando la eficiencia y el control del negocio.
