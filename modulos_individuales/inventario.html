Inventario

Objetivo

Gestionar de manera correcta los productos e ingredientes con registros, la consulta de las existencias, la actualización de stock por entradas y salidas, y la eliminación de artículos descontinuados de la cafetería.

Usuarios que ocupan el Módulo

  • Administrador

Requerimientos de Administrador

El administrador debe ser capaz de:

  1. Crear registro de producto/ingrediente: ID producto, Descripción, Tipo, Cantidad o Peso, Precio venta, Precio compra, Caducidad, Marca, Id proveedor
  2. Leer existencia: ID Producto, Descripción
  3. Actualizar inventario: ID Producto, Tipo, Cantidad o Peso, Precio venta, Precio compra, Caducidad, Marca
  4. Eliminar productos descontinuados: ID Producto
