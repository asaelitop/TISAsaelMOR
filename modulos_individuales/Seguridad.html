Seguridad

Objetivo

Eestablecer un sistema de respuesta rápida para la cafetería, permitiendo la activación inmediata de protocolos de emergencia y la documentación detallada de incidentes mediante el monitoreo de las cámaras, el registro de alertas por pánico y la gestión de reportes, se busca mitigar riesgos para proteger la integridad física de clientes y empleados.

Usuarios que ocupan el Modulo

  • Administrador
  • Empleados

Requerimientos de Administrador

El administrador debe de ser capaz de:

    Crear un registro de una nueva camara de seguridad y un contacto nuevo de emergencia, o generar un reporte de forma manual si es que no se activó el boton de panico:

  • Tabla Camaras_Seguridad: id_camara, ubicacion, estado_operativo, fecha_instalacion
  • Tabla Contacto_Emergencia: id_contacto_emergencia, servicio, numero_telefonico, relacion_tipo_incidente
  • Tabla Incidentes: fecha_hora_alerta, tipo_incidente, ubicacion_alerta, descripcion_preliminar, id_camara_relacionada.
  • Tabla Reportes_Incidentes: id_incidente, descripcion_detallada, acciones_tomadas, personal_responsable, estado_reporte y fecha_cierre_reporte.

    Leer todos los datos registrados de las tablas:

  • Tabla Camaras_Seguridad: id_camara, ubicacion, estado_operativo, fecha_instalacion.
  • Tabla Contacto_Emergencia: id_contacto_emergencia, relacion_tipo_incidente.
  • Tabla Incidentes: id_incidente, tipo_incidente
  • Tabla Reportes_Incidentes: id_reporte, id_incidente

    Actualizar datos registrados para las tablas Camaras_Seguridad y Contacto_Emergencia, y si el administrador levanta un reporte de forma manual se pueden actualizar la tabla Incidentes y Reportes_Incidentes:

  • Tabla Camaras_Seguridad: id_camara, ubicacion, estado_operativo, fecha_instalacion
  • Tabla Contacto_Emergencia: id_contacto_emergencia, servicio, numero_telefonico, relacion_tipo_incidente
  • Tabla Incidentes: fecha_hora_alerta, tipo_incidente, ubicacion_alerta, descripcion_preliminar, id_camara_relacionada.
  • Tabla Reportes_Incidentes: id_incidente, descripcion_detallada, acciones_tomadas, personal_responsable, estado_reporte y fecha_cierre_reporte.

    Eliminar datos registrados para las tablas Camaras_Seguridad y Contacto_Emergencia, y si el administrador levanta un reporte de forma manual se pueden actualizar la tabla Incidentes y Reportes_Incidentes:

  • Tabla Camaras_Seguridad: id_camara
  • Tabla Contacto_Emergencia: id_contacto_emergencia
  • Tabla Incidentes: id_incidente
  • Tabla Reportes_Incidentes: id_reporte
