Seguridad

Objetivo

Proteger la información y accesos del sistema a través de la creación, consulta, modificación y eliminación de usuarios con distintos niveles de permisos con el propósito de garantizar la integridad de los datos y evitar accesos no autorizados.

Usuarios que ocupan el Modulo

  • Administrador

Requerimientos de Administrador

El administrador debe de ser capaz de:

  1. Crear usuarios con roles y permisos: ID Personal, Nombre, Contraseñas, Rol
  2. Leer accesos y registros de actividad: ID Personal, Nombre, ultimo_inicio_sesion
  3. Actualizar contraseñas y privilegios: ID Personal, Contraseñas, Rol
  4. Eliminar cuentas inactivas: ID Personal, Último inicio de sesión
